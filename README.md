# MSCH (Period Tracker)

A minimalist Android application for tracking menstrual cycles and analyzing cycle statistics.

## Technical Stack
* **UI**: Jetpack Compose with Material 3
* **Architecture**: MVVM
* **Database**: Room Persistence Library
* **Language**: Kotlin (Coroutines, Flow)
* **Localization**: Full English and Russian support

## Key Features
* **Cycle Tracking**: Logging period dates and viewing current cycle status on the main dashboard.
* **Statistics**: Calculation of average cycle length, period duration, and variability range.
* **Predictions**: Forecasting upcoming cycles based on historical data.
* **Notifications**: Customizable reminders for upcoming and start-day alerts.
* **Data Management**: Export and import options via JSON and CSV, including a plain-text import tool.
* **Theming**: Support for Light, Dark, and System Default themes.

## Usage
The latest version of the application is available in the Releases section. Download and install the APK on your Android device.
